# AzureAD-Auth
Followed a microsoft tutorial for authentication with Azure AD (Only works if you create an app in AzureAD and do the proper config)

## Project setup

```
cd SPA
npm install
cd API
npm install
```

### Compiles and hot-reloads for development
```
cd SPA
npm start
cd API
npm start
```
